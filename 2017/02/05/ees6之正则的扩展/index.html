字符串的正则方法

字符串的四个方法：

  • search(reg)：如果匹配成功，返回匹配成功的正则表达式第一次出现在字符串中的位置，如果匹配不成功，返回-1
  • match(reg)：匹配正则表达式，并以数组的形式返回匹配的形式,没有返回null
  • replace(regexp|substr,newSubStr|func): 使用替换值替换掉匹配模式在原来字符串中的某一项或所有，返回替换后的新的字符串
  • split([separator][, limit]): 将字符串拆分为子字符串，并以数组的形式返回

es6将四个方法在内部全部调用RegExp的实例方法

  • String.prototype.match 调用 RegExp.prototype[Symbol.match]
  • String.prototype.replace 调用 RegExp.prototype[Symbol.replace]
  • String.prototype.search 调用 RegExp.prototype[Symbol.search]
  • String.prototype.split 调用 RegExp.prototype[Symbol.split]

y修饰符

粘连（sticky）修饰符,表示后一次的匹配目标必须紧随前一次匹配项,多次匹配时，每次的匹配目标必须在起始位置，后一次匹配必须在前一次匹配成功的下一个位置进行。g修饰符只要剩余位置中存在匹配即可，y修饰符确保匹配必须从剩余的第一个位置开始

y修饰符号隐含了头部匹配的标志^

当y和全局g同时出现时，全局匹配将会受限，gy组合还会对String的replace和match两个方法起作用，全局匹配g加上y之后，匹配模式更严格了一些，每一次匹配时目标必须出现在首位：
