es6之Symbol

  • es6

Symbol
原始数据类型，表示独一无二的值

Symbol函数不能使用new命令，他是原始数据类型
Symbol可以接受一个字符串，表示对Symbol实例的描述

如果Symbol参数是一个对象，先调用toString转换为字符串，在生成一个Symbol值

Symbol的参数只表示对当前的Symbol值的表述，所以相同参数的Symbol函数的返回值不相等;

Symbol值不能与其他类型进行运算

Symbol可以显式转换为字符串

作为属性名的Symbol

  • Symbol作为对象的属性名时不能用点运算符
  • Symbol值作为属性名时，该属性是公开的，不是私有的

消除魔术字符串

魔术字符串：在代码之中多次出现、与代码形成强耦合的某一个具体的字符串或者数值

属性名遍历

Symbol作为值不能被常规的方法遍历得到，所以可以为对象定义一些非私有的、但又希望用于内部的方法

Symbol.for/Symbol.keyFor

Symbol.for:接受一个字符串，搜索有没有以该参数为名称的Symbol值，有?返回这个Symbol值:新建并返回一个以字符串为名称的Symbol值。
Symbol.for会被登记在全局环境中供搜索,是全局变量,可以在不同的iframe或service worker中取到同一个值

Symbol.keyFor:为Symbol值登记名字，

模块的 Singleton 模式

使用Symbol

内置的Symbol值

  • Symbol.isConcatSpreadable: 等于一个布尔值，表示该对象使用Array.prototype.concat()时，是否可以展开,类数组对象也可以使用

对于一个类来说，Symbol.isConcatSpreadable属性必须写成实例的属性。

  • Symbol.species:指向当前对象的构造函数，默认会调用这个方法，即使用这个属性返回的函数当作构造函数，来创造新的实例对象。

  • Symbol.iterator: 指向该对象的默认遍历器方法
